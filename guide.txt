1) Graphics

X - you
. - your path
o - the grass
~ - the sea, you can't step here
T - the tree, you can't step here
E - an enemy zone, enter it if you want to fight

2) Controls

Map:
WASD to move

Battle:
F to attack
H to heal
L to leave battle(what a shame)

3) settings.ini guide

!!Default config is great!!

Map Scale - a map scale(AxA), format(1)
Trees Spawn Chance - chance to spawn trees, format(1)
Path - leave a path after your steps(True if enabled, anyting other if not)
Show Enemies - show enemies with the map(True if enabled, anyting other if not)

Hero HP - hero's HP, format(1)
Hero Attack - hero damage, format(min,max) you can use the same dmg if u want for example(10,10)
Hero Heal - hero heal, format(min,max) you can use the same heal amount if u want for example(10,10)
Imp HP - imp hp, format(min,max) you can use the same HP if u want for example(10,10)
Imp Attack - imp damage, hero attack, format(min,max) you can use the same dmg if u want for example(10,10)
Enemies Count - enemies(but not bigger than map scale^2), format(1)
Restore Health After The Battle - when the battle ends, your health gets restored(True if enabled, anyting other if not)
